# Operix Inventory System

This project is an inventory management system for a printing service.  
It consists of:

- **Backend**: Node.js + Express + PostgreSQL (Neon)
- **Frontend**: React
- **Database**: PostgreSQL (hosted on [Neon](https://neon.tech/))

---


## Project Structure
Operix/
├─ backend/
│  ├─ frontend/       # React app
│  ├─ db.js           # PostgreSQL connection
│  └─ server.js       # Express backend
├─ README.md
